CleanCodeSaver - README
=======================

The screen saver is configured via a XML-file. The location for the file "CleanCodeSaver.xml" 
depends on the Windows version:

   32 Bit Windows  : %SystemRoot%\system32\CleanCodeSaver.xml
   64 Bit Windows  : %SystemRoot%\SysWOW64\CleanCodeSaver.xml
   
The installation directory contains a XML schema file that can be used to create an individual
error free configuration file.
